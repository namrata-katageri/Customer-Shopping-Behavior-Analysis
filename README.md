# Customer Shopping Behavior Analysis

## 📌 Project Overview
This project explores customer shopping patterns using Python (Jupyter Notebook), SQL queries, and Power BI dashboards.  
The goal is to analyze customer behavior, identify trends, and generate actionable insights for business decision-making.

---

## 🎯 Objectives
- Analyze customer shopping behavior using transactional data.
- Identify top products, customer segments, and seasonal trends.
- Build interactive dashboards for visualizing KPIs.
- Apply SQL queries for structured data analysis.
- Use Python for preprocessing, statistical analysis, and visualization.

---

## 🛠 Tech Stack
- **Python (Jupyter Notebook)** – Data cleaning, preprocessing, and exploratory analysis.
- **SQL** – Querying and extracting insights from structured datasets.
- **Power BI** – Interactive dashboards and KPI visualization.
- **Excel/CSV** – Dataset storage and validation.

---

## 📊 Key Features
- Customer segmentation and shopping trend analysis.
- SQL queries for product and revenue insights.
- Python notebooks for statistical analysis and visualization.
- Power BI dashboards for interactive reporting.
- End-to-end ETL workflow: extract, transform, and load customer data.

---

## 📈 Insights & Impact
- Identified shopping behavior patterns across different customer groups.
- Highlighted top-selling products and seasonal demand.
- Delivered dashboards that support data-driven decision-making.
- Improved visibility into customer preferences and revenue drivers.
